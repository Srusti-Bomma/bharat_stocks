About Bharat Stocks

Project overview, technical details, and team information

Project Overview

Bharat Stocks is a comprehensive FastAPI backend with a static frontend solution designed specifically for the Indian stock market. Our platform provides real-time access to NSE (National Stock Exchange) stock data, advanced analytics, and intelligent market insights.

Built with modern web technologies and scalable architecture, Bharat Stocks Insight delivers a seamless experience for both retail investors and financial professionals. We combine live market data, historical analysis, and trending insights to empower users with actionable intelligence.

Key Features

Live Market Data

Real-time NSE stock quotes, price movements, and volume analysis with sub-minute updates.

Trending Analysis

Identify market trends, gainers, losers, and sector performance with advanced algorithms.

Most Active Stocks

Track the most actively traded stocks by volume and trading frequency across NSE.

Market News

Integrated news feed from reliable sources to keep you updated with market-moving events.

Admin Dashboard

Comprehensive admin panel for system monitoring, user management, and analytics.

Data Analytics

Historical data storage, performance metrics, and intelligent data ingestion system.

Technical Architecture

Backend Technology

  • FastAPI - High-performance web framework for APIs
  • SQLAlchemy - Powerful Python SQL toolkit and ORM
  • MySQL - Robust relational database management
  • Pydantic - Data validation using Python type hints
  • JWT - Secure authentication and authorization
  • Uvicorn - Lightning-fast ASGI server

Frontend Technology

  • HTML5 - Modern semantic markup
  • Tailwind CSS - Utility-first CSS framework
  • Vanilla JavaScript - Clean, modern JavaScript
  • Google Auth - Secure OAuth2 integration
  • Inter Font - Modern, legible typography
  • Responsive Design - Mobile-first approach

Data Sources & APIs

Stock Data Provider

Indian Stock API - Real-time NSE data with comprehensive coverage of Indian equity markets.

News Integration

GNews API - Curated financial news and market updates from reliable sources.
